if / elif / else

📌 if → testa a primeira condição.
📌 elif → testa outras condições se o if (ou outro elif) for falso.
📌 else → executa se nenhuma condição anterior for verdadeira.

Só pode ter um if, vários elif e um else no final


    idade = 17
if idade >= 18:
    print("Maior de idade")
elif idade >= 16:
    print("Adolescente")
else:
    print("Menor de idade")

for

📌 É um laço de repetição (loop).
📌 Serve para repetir um bloco de código para cada item de
uma sequência (lista, string, range, etc.).

📌 "range" é só um jeito prático de gerar números em sequência.
📌 O "for" pode rodar em qualquer coisa que seja iterável:
listas, strings, tuplas, dicionários, sets… 


    for i in range(1, 6):
    print(i)
    
    #vai imprimir:
    1
    2
    3
    4
    5

while

📌 Diferente do for (que percorre elementos de uma sequência),
o while repete enquanto a condição for verdadeira.

📌 Enquanto a condição for True, o código repete.
Quando virar False, o loop para. 


    
contador = 2
 while contador <= 5:
    print(contador)
    contador += 2
  
  #Soma +2 → vira 4 → ainda é <= 5, imprime.

  #Soma +2 → vira 6 → não é <= 5, então para.

Funções

📌 Uma função é um bloco de código que você pode definir uma vez e reutilizar sempre que precisar.
📌 Serve para organizar o código, evitar repetição e deixar tudo mais legível.

📌 def nome(): → cria a função.
📌 nome() → chama a função.
📌 parâmetros → valores que você passa pra dentro.
Difereça do print pro retuno.

📌 print → mostra na tela (exibição).
📌 return → devolve valor para o programa (uso lógico) 


    3 Tipos principais de funções
    
    #Sem parâmetro, sem retorno
    
    def mensagem():
    print("Olá!")
    
    mensagem()
    
    Print:
    Olá!
    
    #Com parâmetro, sem retorno
    
    def saudacao(nome):
    print("Olá,", nome)
    
    saudacao("Ana")
    
    Print:
    Olá, Ana
   
    #Com parâmetro e com retorno
    
    def soma(a, b):
    return a + b

    print(soma(3, 5)

Listas

📌 É uma coleção de valores dentro de colchetes [ ].
📌 Pode guardar números, textos, ou até misturar tipos diferentes.
📌 Os elementos ficam em ordem e podem ser repetidos. 


    lista inicial
    
nomes = ["joaquim","Maria","Ana"]
print("lista inicial:",nomes)

# Adicionadno elementos
    
nomes.append("Carlos")      # Adicionando ao final
print("Após append:",nomes)

nomes.insert(1,"Fernanda")     # Isere "Fernanda" na posição 1
print("Após insert:", nomes)

# Modificando elementos
    
nomes[2] = "Paulo"     # Modifica o elemento no índice 2
print("Após modificação:", nomes)

# Removendo elementos
    
del nomes[3]      # Remove o elemento da índice 3
print("Após del:", nomes)

nomes.remove("Maria")     # Remove a primeira ocorrência da "Maria"
print("Após remove:", nomes)

Removido = nomes.pop(2)     # Remove e retorna o elemento no índice 2
print(f"Após pop (removido) '(removido)'):", nomes)

nomes.clear()    # Esvazia a lista
print("Após clear:", nomes)

             RESUMO

               

#append() → adiciona no final.

#insert() → insere em uma posição específica.

#atribuição [] = → modifica um valor.

#del → remove pelo índice.

#remove() → remove pelo valor.

#pop() → remove pelo índice e retorna o valor removido.

#clear() → esvazia toda a lista.

📌 except → o que fazer se der erro.
📌 else → roda se não der erro.

F-strings

📌 {x:.1f} → 1 casa decimal
📌 {x:.2f} → 2 casas decimais
📌 {x:.0f} → arredonda sem casas decimais

📌 O f-string formata para 1 casa decimal → 7.6. 


media = 7.567
print(f"Média final: {media:.1f}")

With

📌 with = abre um recurso (arquivo, conexão, etc).
📌 Quando termina o bloco → ele fecha sozinho.

Modos principais de abrir arquivo no python:

📌 "r" → read → só leitura (padrão).
📌 "w" → write → escrita (apaga tudo e escreve do zero).
📌 "a" → append → escrita acrescentando no final (não apaga o que já existe).
📌 open('arquivo.text','r'): abre o arquivo no modo leitura ('r') 


nome = input('Digite o nome: ')
email = input('Digite o email: ')

with open('pessoa.text','a') as arquivo:
    arquivo.write(nome + ' | ' + email + '\n')
